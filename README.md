Getting Started
To get a local copy up and running follow these simple example steps.

- ### Pre requisites

  - npm
  - npm install npm@latest -g
  - Installation
  - Clone the repo
  - git clone <https://github.com/danfranco73/ecommerceDanFranco.git>
  - Install NPM packages
  - npm install
  - Run the APP
  - npm start;
